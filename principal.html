Explicação Listas

Listas Ordenadas

voltar ao início

As listas ordenadas são criadas com a tag ol. Cada item da lista é criado com a tag li. Você pode usar o atributo type para alterar o tipo de numeração da lista. Por exemplo, você pode usar type="A" para usar letras maiúsculas em vez de números.

Listas Não Ordenadas

voltar ao início

As listas não ordenadas são criadas com a tag ul. Cada item da lista é criado com a tag li. O estilo padrão é uma lista com pontos, mas você pode usar o atributo type para alterar o estilo. Por exemplo, você pode usar type="square" para criar uma lista com quadrados.

Explicação H1,H2 ...

voltar ao início

As tags HTML h1, h2, h3, h4, h5 e h6 são usadas para definir títulos e subtítulos em HTML.
Elas são usadas para separar o conteúdo em seções e subseções, e são organizadas em ordem hierárquica, onde h1 é o título principal e h6 é o título menos importante
